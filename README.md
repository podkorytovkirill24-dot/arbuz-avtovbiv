# Avtovbiv Bot

Бот для сдачи номеров через личку и выдачи в рабочие группы, с очередью, статусами и большой админ‑панелью.

## Установка

1. Установите Python 3.10+.
2. Установите зависимости:

```bash
pip install -r requirements.txt
```

3. Настройте переменные окружения (рекомендуем через `.env`):

```bash
# .env
BOT_TOKEN=ВАШ_ТОКЕН
ADMIN_IDS=8500766185
```

Либо задайте их на время сессии:

```bash
set BOT_TOKEN=ВАШ_ТОКЕН
set ADMIN_IDS=8500766185
```

Можно указать несколько админов через запятую, например: `ADMIN_IDS=8500766185,123456789`.

4. Запуск:

```bash
python main.py
```

После первого запуска появится база `bot.db`.

## Настройка в Telegram

1. В личке боту напишите `/admin` и настройте тарифы, офисы и приемки.
2. Добавьте бота в группу приемки (куда сдают номера) и дайте ему права администратора.
3. В группе приемки выполните `/num` и выберите тариф.
4. Добавьте бота в рабочую группу (выдача номеров) и дайте ему права администратора.
5. В рабочей группе выполните `/set` и выберите офис для привязки.
6. Пользователи могут сдавать номера в личке через меню или в группе приемки.
7. Операторы в рабочей группе пишут слово `номер` — бот выдаёт следующий номер с кнопками статусов.

## Команды

- `/start` — открыть главное меню.
- `/admin` — админ‑панель (только для админов).
- `/num` — привязать тариф к группе приемки.
- `/set` — привязать офис к текущей группе/теме.

## Важно

- В группах лучше отключить privacy mode для бота через BotFather или выдать права администратора.
- Токен из переписки публичный — обязательно перевыпустите токен в BotFather и замените переменную `BOT_TOKEN`.
